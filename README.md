This is a discord bot for scheduling the Dune classic board game.

Right now it simply allows you to start the faction selection process.

Faction selection process:
1. Select factions you want to draw from
2. Given an ordered list of users in a game (first come, first-served) (max of 5 players)
3. Draw 3 random factions from pool of available factions
4. Allow player 1 to choose from the 3 factions - If a player wishes to read details on a faction then they can and they will be given a link to download the pdfs of the rules provided
5. A faction is chosen from the 3
6. The 2 factions left over go back into the pool of available factions
7. The chosen faction is no longer available for draw
8. Display player name and faction chosen on the board
9. Repeat steps 3 - 8 until final player has chosen
10. Once all players have chosen display a final list of all player names and the faction they have chosen
